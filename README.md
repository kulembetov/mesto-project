## Название проектной работы
[Место](https://kulembetov.github.io/mesto-project)

## Описание
Одностраничный сайт, был свёрстан по макету из Figma и адаптирован под различные устройства (320px и 1280px). На котором, пока что можно просматривать фотографии различных мест. Но в будущем будет возможность добавлять свои, обновлять изображение профиля и ставить лайки. Сделан в рамках учёбной деятельности в [Яндекс.Практикуме](https://practicum.yandex.ru) на факультете ["Веб-разработчик плюс"](https://practicum.yandex.ru/web-plus).
